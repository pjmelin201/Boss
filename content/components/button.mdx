### Leading and trailing visuals

    />
  </Box>
  <Box>
    <Heading sx={{fontSize: 3}}>Trailing action</Heading>
    <Box as="p">
      Trailing actions such as dropdown carets are always locked to the end of a button, which is particularly obvious
      for full width scenarios.
    </Box>
    <Box as="p">
      If a button is a call-to-action (Submit, Save, Do this™, etc), its contents (visuals + button label) appear
      center-aligned inside the button container.
    </Box>
    <Box as="p">
      If a button is used for selecting items (Weeks ▾, Iteration ▾, Sort ▾, Select™ ▾, etc), its contents (visuals +
      button label) appear left-aligned inside the button container.
    </Box>
  </Box>
</Box>
